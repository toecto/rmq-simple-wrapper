# rmq-simple-wrapper
AMQP, RabbitMQ simplified client

## Installation with composer:

```javascript
    "repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:toecto/rmq-simple-wrapper.git"
        }
    ],
    "require": {
        "toecto/rmq-simple-wrapper": "dev-master"
    }
  ```
